# Shared Knowledge

Reusable reference material for the Nexa Product Pipeline module.

- `reference/` contains evergreen module knowledge.
- `how-to/` contains repeatable cross-module procedures.
- `stack/` contains implementation stack notes once a stack exists.
- `tooling/` contains tool and workflow notes.

Keep module-build facts in `projects/nexa-productpipeline/`. Promote only durable, reusable findings into `shared/`.
